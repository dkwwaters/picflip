PicFlip — Support

PicFlip is a batch image converter for the Mac: drop in HEIC photos (or any images, or whole folders), pick a format, and flip them all in one go. You can also import photos directly from a USB-connected iPhone or iPad.

Frequently asked

My iPad/iPhone doesn't show up in the import window.
Connect it with a USB cable, unlock it, and tap "Trust This Computer" if asked. If it still doesn't appear, try a different cable or port.

Where do converted files go?
Next to the originals by default, or into any folder you choose. PicFlip never overwrites or deletes your original photos.

A file says "No permission to write here".
macOS grants access to what you add. If you dragged in a single file, PicFlip can't write beside it — choose a destination folder instead, or drag the whole folder in.

Why isn't WebP in the format list?
The format list shows what your version of macOS can encode. WebP appears on Macs that support it.
